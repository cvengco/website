Variables

const vs let

Declare Variable using `var`, `let`, `const`, or nothing.

let

`var` was using to declare variables prior to 2015. `let` and `const` are used to declare variables after 2015. Older browsers will only support use of `var`.

`let` allow you to define a variable that can have its value changed.

`const` declares a constant value that cannot be changed.

It is recommended to use either `let` or `const`.

Multiple variables can be declared using one statement:
`let person = "John Doe", carName = "Volvo", price = 200;`
